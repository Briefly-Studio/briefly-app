# Briefly

Briefly is a focused flashcard and study app designed around clarity, trust, and intentional learning.  
The product prioritizes signal over noise and avoids premature analytics or automation.

---

## Current Status

**Active Development:** v2.x  
**Latest Frozen Release:** v1.5.0  
**Platform:** React Native (Expo)

> v1.5 is considered stable and frozen.  
> All new work occurs under v2.x.

---

## Philosophy

Briefly is built on a few core principles:

- Learning tools should reduce cognitive load, not increase it
- Data should be meaningful before it is visualized
- Users should understand *why* the app behaves the way it does
- Trust and portability matter more than novelty

---

## Features (v1.5 – Frozen)

- Flashcard decks with manual card management
- Review mode with session timing
- Quiz mode with scoring and percentage results
- Session history (review + quiz)
- Lightweight stats (no charts by design)
- Local-first storage
- Predictable navigation and completion flows

---

## Roadmap (v2.x)

### v2.1 – Difficulty Tagging (Manual)
- Card-level difficulty tags (easy / medium / hard)
- Used as learning signals (not automation yet)

### v2.2 – Smart Shuffle
- Weighted review order based on difficulty and recency
- Optional toggle per deck

### v2.3 – Deck Export
- JSON export (full fidelity)
- CSV export (cards)
- PDF export (read-only)

> Automation and analytics will only be added once sufficient, trustworthy data exists.

# Briefly v1 — Manual Study MVP (Completed)

## Scope
- Offline-first
- Local-only storage (AsyncStorage)
- No authentication
- No AI
- No cloud sync

## Features
- Deck creation & deletion (with cascade delete)
- Card CRUD per deck
- Review mode (flip cards)
- Quiz mode (MCQ, scoring, retry)
- Expo Router navigation

## Architectural Notes
- Deck → Card parent-child relationship
- Deterministic storage keys
- No circular storage imports
- Clean separation of concerns
